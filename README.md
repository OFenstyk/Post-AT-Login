## Getting Started

### Prerequisites

- Node.js
- Playwright
- VSCode

### Installation

# Clone the repository

````bash
git clone https://github.com/OFenstyk/Post-AT-Login.git

# Change into the project directory
cd Post-AT-Login

# Install dependencies
npm install
````

## Usage

````bash
# Run the tests
npm test
````
